# 🟣 Angular Admin Dashboard (Fast Track Project)

A modern Angular-based Admin Dashboard built with a fast-track learning approach.  
This project focuses on Angular architecture, services, routing, authentication, RxJS state management, and real-world UI patterns.

---

## 📌 Features

- 🔐 Authentication System (Login / Register UI)
- 🧭 Role-based Access (Admin / User)
- 📊 Admin Dashboard Layout
- 👤 Users Management (CRUD)
- 📝 Posts Management (CRUD)
- 🔍 Search & Filtering
- 📄 Pagination Support
- 🧠 RxJS-based State Management
- ⚡ Lazy Loaded Routes
- 🛡️ Route Guards (Protected Routes)

---

## 🛠️ Tech Stack

- Angular 17+
- TypeScript
- RxJS
- Angular Router
- HttpClient
- SCSS / CSS
- (Optional) Angular Material / Tailwind

---

## 📁 Project Structure

```

src/
````

## ⚙️ Key Concepts Implemented

### 🧩 Angular Architecture
- Component-based structure
- Services for business logic
- Dependency Injection (DI)

### 🔄 State Management
- RxJS Observables
- BehaviorSubject for shared state

### 🔐 Authentication Flow
- JWT-based login (frontend handling)

- HttpClient for backend communication
- Centralized API service layer
